# 🍽️ Recipe Guide App
 
A desktop application for browsing and managing recipes, built with **C#** and **Windows Forms** in **Visual Studio**, using a local **SQLite** database for data storage.
 
## 🛠️ Tech Stack
 
- **Language:** C#
- **UI Framework:** Windows Forms
- **IDE:** Visual Studio
- **Database:** SQLite
## ✨ Features
 
- Browse and search recipes
- View detailed recipe information
- Add, update, and delete recipes
- Local data persistence via SQLite
## 🚀 Getting Started
 
### Prerequisites
 
- Visual Studio (2019 or later recommended)
- .NET Framework / .NET (compatible version)
### Installation
 
1. Clone the repository:
```bash
   git clone https://github.com/OsmanGundogdu/Recipe_Guide_App.git
```
2. Open `Yazlab1.sln` in **Visual Studio**.
3. Build the solution (`Ctrl + Shift + B`).
4. Run the application (`F5`).
> The SQLite database file (`TarifRehberiUygulamasi.db`) is included in the repository and will be used automatically on startup.
